# 🧩 Código Secreto

Um jogo simples de adivinhação de código feito com **HTML, CSS e JavaScript**.  
O objetivo é descobrir o código secreto dentro de diferentes níveis de dificuldade.

---

## 🚀 Demonstração

O jogador faz login, escolhe o nível de dificuldade e tenta acertar o código secreto através de palpites.

---

## 🎮 Funcionalidades

- 🔐 Sistema de login simples
- 🎯 Jogo de adivinhação de código
- 📊 Níveis de dificuldade:
  - Fácil
  - Médio
  - Difícil
  - Impossível
- 🔄 Botão de reiniciar partida
- 🎉 Popup de vitória
- 🧠 Interface interativa com feedback ao jogador
- 📜 Registro de tentativas (log)

---

## 🧱 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Como jogar
Faça login: admin e senha: 1234
Escolha um nível de dificuldade
Digite seu palpite no campo indicado
Receba dicas e tente descobrir o código secreto
Vença e veja a mensagem de vitória 🎉
